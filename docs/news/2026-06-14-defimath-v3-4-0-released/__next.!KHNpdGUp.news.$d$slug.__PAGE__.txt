<h2>Foundry property-fuzz layer</h2>
<p>Hardhat-only testing validates each function at concrete points against
external JS references. Foundry adds a complementary layer that validates the
<strong>algebraic structure</strong> across the random input space.</p>
<p>92 properties organized into five categories — round-trips, monotonicity,
identities, output bounds, symmetries. Per-module breakdowns on each
module's <a href="/docs#testing">docs page</a>. Full suite completes in ~6 seconds
locally and ~30 seconds on a 2-core CI runner.</p>
<p>Both layers run via <code>npm test</code> — Hardhat first (correctness + gas regression),
then Foundry (algebraic structure).</p>

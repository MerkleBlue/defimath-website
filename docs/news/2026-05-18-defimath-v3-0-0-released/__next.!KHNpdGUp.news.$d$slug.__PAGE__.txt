4:T896,<p>A major release that extends DeFiMath beyond options pricing into general
portfolio and performance math — useful for vaults, structured products,
risk engines, and any protocol that needs to reason about returns on-chain
— and ships under a fresh Solidity version with full test coverage.</p>
<h2>Statistics</h2>
<ul>
<li><code>mean</code> — arithmetic mean, Σ(vᵢ) / n.</li>
<li><code>geometricMean</code> — geometric mean of two values, √(a · b).</li>
<li><code>weightedAverage</code> — weighted average, Σ(vᵢ · wᵢ) / Σ(wᵢ).</li>
<li><code>stdDev</code> — sample standard deviation, √(Σ(vᵢ − μ)² / (n − 1)).</li>
</ul>
<h2>Risk and return</h2>
<ul>
<li><code>historicalVolatility</code> — annualized volatility from a price series
(sample std dev of log returns, annualized).</li>
<li><code>sharpeRatio</code> — risk-adjusted return: (annualized return − risk-free
rate) / annualized volatility.</li>
<li><code>maxDrawdown</code> — largest peak-to-trough decline on an equity curve,
expressed as a fraction.</li>
<li><code>valueAtRisk</code> (VaR) — historical Value at Risk: the return threshold
below which losses fall with probability 1 − α.</li>
<li><code>conditionalValueAtRisk</code> (CVaR) — Expected Shortfall: the mean return
in the left tail beyond VaR.</li>
<li><code>yieldToMaturity</code> — closed-form YTM for a zero-coupon bond,
<code>ln(faceValue / price) / timeYear</code>, continuous compounding.</li>
<li><code>internalRateOfReturn</code> — IRR via Newton-Raphson; solves
Σ Cᵢ · e^(−irr·tᵢ) = 0 from an initial guess.</li>
</ul>
<h2>Release polish</h2>
<ul>
<li>Bumped to <strong>Solidity 0.8.35</strong> (also remains compatible down to 0.8.31)</li>
<li>Compiler warnings cleaned up</li>
<li>Output parameters now have explicit names across the API</li>
<li>NatSpec author tags added throughout</li>
<li>Repository folder structure tightened</li>
<li><strong>100% test coverage</strong> — every public function, every branch</li>
</ul>

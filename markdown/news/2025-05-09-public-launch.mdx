The DeFiMath Solidity library is now open for public use. Our initial launch
includes core math and derivative functions for safe and optimized operations,
headlined by a gas-optimized **Black-Scholes** options pricing implementation
— DeFiMath's main advantage over existing on-chain math libraries.

DeFiMath has been **6 months in the making** — work began in October 2024
and the library has been refined through iterative gas and precision
optimization ever since.

This release lays the groundwork for future DeFi primitives and complex
financial calculations.

Check out the [DeFiMath GitHub repo](https://github.com/MerkleBlue/defimath).

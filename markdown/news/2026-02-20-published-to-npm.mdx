---
title: DeFiMath is now on npm
date: 2026-02-20
excerpt: The DeFiMath Solidity library is now installable as an npm package.
---

DeFiMath is now [published on npm](https://www.npmjs.com/package/defimath-lib)
and can be installed as a regular package alongside your other dependencies.

```
npm install defimath-lib
```

The README has been updated for npm consumers and unused dependencies have
been pruned from the package.

See the [DeFiMath GitHub repo](https://github.com/MerkleBlue/defimath) for
full installation and usage details.
